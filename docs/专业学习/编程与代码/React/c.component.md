# 3.创建组件

在react中，我们可以使用`jsx`中的一个`js`函数来创建组件。  

需要注意的是，在js函数返回的html代码中，有且只能有一个父组件，例如，下面这个同时两个组件，就是不合法的。  

```js
return (
    <Header/>
    <Footer/>
);
```

但是，如果必须要多个组件并列，该如何解决呢？——可以通过用一个空标签包围。下面这个返回就是合法的。  
```js
return (
    <>
        <Header/>
        <Footer/>
    </>
);
```

在编写好函数后，我们使用下面的语句将组件函数以标签的形式导出：  
```js
export default <functionName>
```
