# 欧几里得算法

又名辗转相除法，是求最大公约数的一种算法。  

具体来说，若有 \( a \) 、 \( b \) 两个数，且 \( a>b \)，则 \( a \div b \) 所得余数 \( c \) 与 \( b \) 的公约数和 \( a \) 、 \( b \) 的公约数相同。所以，若求两数的最大公约数，只需要辗转相除直至余数为0，则除数既是最大公约数。  

实现：  

```python
def gcd(a , b):
    if a < b:
        a , b = b , a
    while b > 0:
        a = a % b
        a , b = b , a
    return a
```
