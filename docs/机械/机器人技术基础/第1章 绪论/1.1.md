# 机器人的概念

## 机器人的基本结构

机器人的基本结构：  

- **执行机构**：对应人的躯干和四肢  
- **驱动-传动系统**：对应人的肌肉、肌腱  
- **控制系统**：对应人的大脑  
- **感知反馈系统**：对应人的感觉器官、神经等  

## 机器人的子系统

机器人系统通常包括3个子系统，分别是：  

- 运动子系统：Motion subsystem
- 辨识子系统：recognition subsystem
- 控制子系统：control subsystem

**运动子系统**：用于实现期望运动的执行机构  

**辨识子系统**：使用一系列传感器来获取机器人本身、操作对象和环境的信息  

**控制子系统**：根据辨识子系统的信息来控制机器人的运动，以完成既定动作任务
