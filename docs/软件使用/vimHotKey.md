# Vim的使用

## 复制、粘贴、剪切

在visual模式下选中文字后：  

- 复制：<kbd>y</kbd>
- 剪切：<kbd>x</kbd>
- 粘贴：<kbd>p</kbd>
- 撤销：<kbd>u</kbd>
- 反撤销：<kbd>Ctrl</kbd>+<kbd>R</kbd>
- 从剪贴板复制：<kbd>"</kbd>+<kbd>+</kbd>+<kbd>y</kbd> (要按shift)
- 从剪贴板粘贴: <kbd>"</kbd>+<kbd>+</kbd>+<kbd>p</kbd>
- 删除： <kbd>d</kbd>
- 跳转到开头：<kbd>g</kbd>+<kbd>g</kbd>
- 跳转到末尾行：<kbd>shift</kbd>+<kbd>g</kbd>
